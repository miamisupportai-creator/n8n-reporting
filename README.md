# Automated Reporting + Analytics

Generates a weekly executive report every Monday at 8am UTC, pulling leads and deals from Zoho CRM, calculating conversion metrics, using Claude to write a narrative summary, and delivering the full report via ntfy.

## Quick Start
1. Download `workflow.json`
2. Import into n8n: **Settings → Import Workflow**
3. Replace `${CLIENT_ID}` and `${CLIENT_NAME}` with your values
4. Configure credentials (Zoho CRM, Claude API)
5. Activate and test
